# 🌐 Modern Fashion / Agency Website

An immersive, animation-driven frontend project inspired by modern fashion and creative agency websites. This project focuses on delivering a visually rich user experience using smooth scrolling, video sections, and clean editorial layouts.

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* GSAP (Animations)
* Locomotive Scroll (Smooth scrolling)

---

## ✨ Features

* 🎥 Video-based hero section for engaging first impression
* 🔄 Smooth scrolling and scroll-triggered animations
* 🎨 Pixel-perfect UI with modern typography and layout
* 📱 Fully responsive across devices
* ⚡ Optimized performance for smooth interactions

---

## 📌 What I Learned

* Implementing advanced scroll-based animations using GSAP
* Creating smooth scrolling experiences with Locomotive Scroll
* Building visually engaging and performance-optimized UI
* Translating modern design concepts into functional frontend
